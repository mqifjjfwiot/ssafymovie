# SSAFYmovie

https://www.themoviedb.org/

TMDB The Movie Api를 활용해 500여개의 영화 데이터를 끌어와 데이터 처리를 해보는 프로젝트입니다.

<h3>참여 인원</h3>
강민구 외 2명. 
  
<h3>프로젝트에서 맡은 역할</h3>
  영화 데이터의 수집, 분류, 형변환.<br>
  필요한 추가 데이터의 정제, 정리<br>
  영화 추천 알고리즘<br>
  연관 영화 분석 알고리즘<br>
  
<h3>데이터 포맷</h3>
  main : json<br>
  sub : csv<br>
